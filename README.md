# 面接練習チャットボット(デモ版)

## 概要
このアプリはIT志望の新卒エンジニアが自己分析を行うためのアプリのデモ版になっています。<br>
質問は3段階となっており、最初に答えた内容を深堀していくようになっています。<br>
最後にフィードバックとして良かった点、改善点、総評などを返します。<br>
回答に対する質問の生成やフィードバックの生成にはOpenAI APIを使用し、LLMに生成してもらいました。<br>
システムプロンプトでは項目ごとに内容を指定し、稼働させながら求めるものに合うように修正を繰り返していきました。<br>
多くの人がより簡単に編集できるように最小限の環境で実装しています。

## 開発の経緯・ストーリー
学校内のローカルLLM用のサーバが用意できたけど使い道が無くて困っているという先生の要望と、<br>
キャリアの授業で行っていた自己分析や面接練習から着想を経て作成に至りました。<br>
このアプリはローカルLLMに接続する前のアプリになっています。

## 機能
- AIによる面接官のロールプレイ（3つの質問とフィードバック）
- 会話履歴の管理（ユーザーIDベース）
- 会話のダウンロード機能

## 技術スタック
- **バックエンド**: FastAPI (Python)
- **AI**: OpenAI API (GPT-4o-mini)
- **フロントエンド**: HTML, CSS, JavaScript (Vanilla)
- **その他**: Pydantic, python-dotenv, uvicorn

## インストール
1. リポジトリをクローンまたはダウンロードします。
2. Python 3.8以上をインストールしてください。
3. 仮想環境を作成し、アクティブ化します：
   ```
   python -m venv .venv
   .venv\Scripts\activate  # Windowsの場合
   ```
4. 依存関係をインストールします：
   ```
   pip install -r requirements.txt
   ```
5. `.env`ファイルを作成し、OpenAI APIキーを設定します：
   ```
   OPENAI_API_KEY=your_api_key_here
   ```

## 使用方法
1. バックエンドを起動します：
   ```
   uvicorn main:app --reload
   ```
   - デフォルトで `http://127.0.0.1:8000` で動作します。
2. ブラウザで `index.html` を開きます（またはローカルサーバーでホスト）。
3. 面接を開始し、質問に答えてください。3問終了後にフィードバックが表示されます。
4. 会話をリセットするには「開始・リセット」ボタンをクリックします。

## APIエンドポイント
- `POST /interview`: ユーザーIDとメッセージを送信してAIの応答を取得します。

## 用途別、プロンプトの編集方法
`main.py`の`SYSTEM_PROMPT`を編集してください
  ```
    SYSTEM_PROMPT= """
      ここのシステムプロンプトを書き換えてください
    """
  ```
## 注意事項
- OpenAI APIキーが必要です。`.env`ファイルに設定してください。
- 本番環境では会話履歴をデータベースに保存することを推奨します。
